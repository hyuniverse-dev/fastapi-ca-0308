User 도메인의 업무(비즈니스)

1. 회원가입 - Create
    - 중복 체크
    - 리소스 식별자(id) 생성
        - 유저의 ID는 임의의 ULID 문자열을 이용한다.
        - 전통적인 UUID와 유사하지만 시간 정보를 포함해서 성능이 더 빠르다.
        - ID를 숫자를 이용한 시퀀스형으로 생성하는 경우도 많지만
          보안적으로 문제가 되기 때문에 잘 사용하지 않는다.
    - 비밀번호 암호화(단방향)
2. 회원탈퇴 - Delete
3. 회원 정보 수정 - Update
4. 회원 정보 조회 - Read(1 건 조회 / 전체 조회)
